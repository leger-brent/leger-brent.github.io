Local Storage

Local Storage is the ability to store web data in the clients web browser. Storage size depends on a browser type and compatibility, but the minimum storage amount is 5MB. Local Storage will keep its data stored forever or until it's explicitly deleted.

Session Storage

Session Storage works exactly the same way as Local Storage with one exception. Session Storage will automatically delete when the session is terminated.

Functions and Uses

There are essentially three functions of Local Storage (these functions apply to Session Storage as well):
  • localStorage.setItem() - which stores information.
  • localStorage.getItem() - which retrieves information.
  • localStorage.clear() - which deletes everything in storage.
An important item to note is that both Local Storage and Session Storage can hold information in a key:value pair. The value cannot be an object, so to store an object, it must first be converted to a JSON string.
