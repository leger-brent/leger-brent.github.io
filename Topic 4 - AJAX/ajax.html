AJAX

AJAX stands for "Asynchronous JavaScript and XML" and is a powerful tool for sending and receiving data in the background without reloading the page.

AJAX makes asynchronous updates by using an XMLHttpRequest objects. These objects can be set to run on a prescribed event such as a page load or button click. The XMLHttpRequest sends either a POST or GET request to another server. These requests track their state changes, as well as responses from the external server.

In my example below, the PokeSelector, I use a script with user input to generate an XMLHttpRequest object to generate a GET request to pokeapi. The script only sends when the Submit button is clicked. Upon the successful response from the API, I parse the incoming JSON and extract the name and types of the Pokemon that corresponds to the number the user has given.

Note that none of the Pokemon data is stored locally, yet I'm able to access over 800 of them using AJAX.
